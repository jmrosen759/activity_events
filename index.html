Using jQuery Selectors and Events

this paragraph will change color when the button is clicked

hover over this box

double click this text and it disappears

Selectors I used

#btn1
This selector uses an id to target the button. When the button is clicked it runs the event attached to it.
#para1
This id selector targets the paragraph so the style can change when the button is pressed.
.box
This is a class selector. I used it to detect when the mouse hovers over the box.
#hideMe
This selector targets the last paragraph. When it gets double clicked the element hides.
